Code Catcher


HOW TO PLAY

The coder needs code to build an awesome website and attract visitors.

He has 1 minute to catch falling pieces of code.

Use the arrow keys to move him left and right.

Every piece of code you catch scores web hits.

= 1 web hit = 3 web hits = 5 web hits

Just watch out for code bugs or you lose 5 web hits!
